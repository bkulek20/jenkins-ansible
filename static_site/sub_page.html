Setting Up the Project Structure and Hosting on GitHub Pages

First, I created a public repository named MyPortfolio2 on GitHub and cloned it to my computer using the following command:
git clone https://github.com/bkulek20/MyPortfolio2

You can view the repository here.

Then, I created a simple index.html file to start building my site.

To publish the project using GitHub Pages, I followed these steps:

• Went to Settings → Pages
• Under Build and deployment, selected Deploy from a branch
• Chose the main branch and saved the changes

After a short delay, the website went live and became accessible through the generated GitHub Pages URL.

I structured my web app as follows:

index.html – frontend
main.css – styling
app.js – functionality

To push changes from VS Code to GitHub, I used the following commands:

git add .
git commit -m "your comment"
git push origin main

Finally, I noticed that updates committed to GitHub took a few seconds to appear on the live site, which is normal with GitHub Pages.
